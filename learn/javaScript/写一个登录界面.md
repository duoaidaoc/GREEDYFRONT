# 创建 React 项目

```bash
# 使用 Vite（推荐）
npm create vite@latest login-app -- --template react
cd login-app
npm install

# 或者使用 Create React App
npx create-react-app login-app
cd login-app
```

# 安装必要的依赖

```bash
# 如果需要路由
npm install react-router-dom

# 如果需要表单处理
npm install formik yup

# 如果需要样式库（可选）
npm install @mui/material @emotion/react @emotion/styled
# 或者
npm install antd
```
